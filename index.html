    2021-03-13 Vehicle Class (Im)Balance

    I my view, the game became more boring after WG added Vehicle Class Matching rules into the MatchMaker (update 5.1). The exact vehicle class matching rules have been adjusted since, but basically the Vehicle Class Matching limits the number of Tank Destroyers and Light Tanks per team and makes sure there is only +/-1 tank difference between the teams across the vehicle classes.

    2020-05-07 BlitzAnalysiz.com launched

    Hy!

    BlitzAnalysiz.com is a site about in-depth game statistics of Wargaming’s World of Tanks Blitz. You won’t find individial player statistics here - for those, please check awesome Blitzstars.com. Instead, the the site is about analysis of the player population and tank performance statistics.
